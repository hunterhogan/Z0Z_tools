# Suggested Commands

- Run targeted tests: `.venv\Scripts\python.exe -m pytest tests\test_autoRevert.py` from repo root.
- Run the full configured test suite: `.venv\Scripts\python.exe -m pytest` from repo root.
- Compile a file: `.venv\Scripts\python.exe -m py_compile Z0Z_tools\autoRevert.py`.
- Search files/text on Windows PowerShell with `rg --files` and `rg -n "pattern"`.
