# Suggested Commands

- Activate the project virtual environment from the repository root before running project commands.
- Run the full configured test suite after every change: `pytest`.
- During development, use targeted pytest commands only as an additional guide, then finish with the full `pytest` suite.
- Compile a changed Python file when syntax feedback is useful: `python -m py_compile <relative-path>`.
- Prefer Serena and other MCP tools for code navigation and symbol lookup before terminal search commands.
