# Core

- Python packages include `Z0Z_tools`, `humpy_toolz`, `humpy_tlz`, and `humpy_cytoolz`; tests live under package `tests` directories and `tests/` where present.
- Public exports are centralized in package `__init__.py` files.
- Most changes to `humpy_cytoolz` are automated, or ought to be automated, from other sources. Unless the user explicitly directs otherwise, do not directly change files in the `humpy_cytoolz` namespace.
- Read `mem:tech_stack` for packaging/runtime details, `mem:conventions` for style, `mem:suggested_commands` for local commands, and `mem:task_completion` before finishing coding tasks.
