# Tech Stack

- Python package, `requires-python >=3.11`, tested against Python 3.11 and 3.14 classifiers.
- Build backend: setuptools via `pyproject.toml`.
- Core dependencies: `numpy`, `scipy`, `resampy`, `soundfile`, `tqdm`, `hunterMakesPy`.
- Optional/testing dependencies include `pytest`, `pytest-xdist`, `pytest-cov`, `torch`, `pytest-mock`.
- Static config files present: `ruff.toml`, `pyrightconfig.json`, `ty.toml`.
